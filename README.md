# Exadata

Exadata es un sistema de informacion desarrollado en Python y MYSQLP que permite el analisis de comentarios en la red social Twitter.

# Modulos
- Administracion de CSV
- Muestra
- Cortar Fecha
- Popularidad
- Nube De Palabras
